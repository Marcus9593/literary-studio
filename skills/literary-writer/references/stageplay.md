# 舞台剧本创作指南

## 核心特点

舞台剧与影视的根本区别：
- **空间限制**：场景转换成本高，需要在有限布景中完成
- **实时性**：没有剪辑，时间是连续的（或通过约定俗成的方式跳转）
- **观演关系**：演员与观众在同一空间，能量直接传递
- **台词为王**：无法用特写镜头，语言承载大部分叙事功能

## 剧本格式

### 基本格式
```
《剧名》

人物：
  张三 — 40岁，公司经理
  李四 — 35岁，张三的妻子
  王五 — 50岁，张三的父亲

时间：当代
地点：某城市

第一幕

第一场

【客厅。傍晚。一张沙发，一张茶几。
张三坐在沙发上，手里拿着一封信。

张三
（读信，声音颤抖）
"亲爱的张三，当你读到这封信的时候……"

李四从厨房走出。

李四
晚饭好了。你在看什么？

张三
（迅速收起信）
没什么。工作的事。
```

### 格式要点
- 人物表在剧本开头
- 舞台指示用方括号【】或斜体
- 角色名居中加粗
- 对白紧跟角色名之后
- 动作/表情提示用括号

## 结构设计

### 独幕剧
- 时长：20-60分钟
- 特点：紧凑、单一冲突、不可分割
- 适合：小型演出、戏剧节、实验性作品

### 多幕剧（标准）
- **两幕制**：最常见的形式
  - 第一幕（约60%）：建立、发展、高潮
  - 中场休息
  - 第二幕（约40%）：解决、结局
- **三幕制**：传统结构，每幕有明确功能
- **多场制**：一场一场推进，适合叙事性强的作品

### 场景转换
- **暗场**：灯光全暗，换景
- **追光**：只照亮特定区域，其他在暗中换景
- **流动布景**：演员或工作人员在光中移动道具
- **空舞台**：极简主义，靠灯光和表演创造空间

## 舞台语言

### 台词特点
- **口语化**：读出来要自然
- **节奏感**：长短句交替，有呼吸感
- **潜台词丰富**：观众通过"没说的"理解"说的"
- **个性化**：每个角色有独特的语言节奏

### 舞台指示
- **简洁**：只写必要的动作和位置
- **开放性**：给导演和演员发挥空间
- **功能性**：说明动作的目的，而非具体执行方式

### 独白与旁白
- **独白**：角色向观众袒露内心（打破第四面墙或内心独白化）
- **旁白**：叙事者功能，跳进跳出故事
- **使用原则**：有戏剧目的，不滥用

## 戏剧冲突

### 冲突类型
- **人与人**：最直接的冲突形式
- **人与自我**：内心挣扎，适合独白展现
- **人与社会**：制度、传统、偏见
- **人与命运**：无法抗拒的力量

### 冲突构建
- 每个场景至少有一个冲突
- 冲突逐步升级
- 高潮是最大冲突的爆发
- 解决方式要出人意料又在情理之中

## 实用技巧

### 开场
- 快速建立情境（谁、在哪、什么关系）
- 引入核心冲突的种子
- 吸引观众注意力的第一个瞬间

### 节奏控制
- 紧张段落后给观众喘息
- 喜剧元素可以穿插在悲剧中
- 静场（沉默）的力量不要忽视

### 结尾
- 有力的最后一句台词
- 留给观众思考空间
- 不要过度解释，信任观众的理解力

### 考虑制作可行性
- 演员数量：越少越容易制作
- 场景数量：越少换景越简单
- 特殊效果：灯光、音效、多媒体的成本
- 时长：90-120分钟（含中场）是商业演出的甜区
