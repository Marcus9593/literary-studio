# 接管质量门控

## 一、实体提取质量

| 指标 | 阈值 | 说明 |
|------|------|------|
| 实体提取置信度 | ≥ 0.7 | 低于此值的实体标记为"待确认" |
| 章节覆盖率 | ≥ 80% | 至少覆盖 80% 的章节 |
| 角色识别完整度 | ≥ 90% | 主要角色不能遗漏 |
| 别名归一准确率 | ≥ 85% | 同一角色的不同称呼应正确合并 |

## 二、结构分析质量

| 指标 | 阈值 | 说明 |
|------|------|------|
| 卷结构识别 | 存在 | 至少识别出卷的划分 |
| 主线识别 | 存在 | 至少识别出一条主线 |
| 高潮分布 | 存在 | 至少标记出一个高潮点 |
| 伏笔识别 | ≥ 50% | 至少识别出一半的伏笔 |

## 三、风格指纹质量

| 指标 | 阈值 | 说明 |
|------|------|------|
| 样本量 | ≥ 5 段 | 至少从 5 个高质量段落提取 |
| 句式分析 | 存在 | 有句式长度分布数据 |
| 对话风格 | 存在 | 有对话特征描述 |
| 叙事视角 | 确定 | 明确是第几人称、什么视角 |

## 四、冲突检测质量

| 冲突类型 | 严重度 | 处理 |
|----------|--------|------|
| 角色能力跳跃 | critical | 必须标记，不能自动忽略 |
| 时间线矛盾 | critical | 必须标记，不能自动忽略 |
| 地点逻辑冲突 | high | 必须标记 |
| 设定自相矛盾 | high | 必须标记 |
| 未闭合伏笔 | medium | 标记，供后续参考 |
| 轻微不一致 | low | 标记，不阻断 |

## 五、接管报告必含字段

### 5.1 项目概览

- 总章节数
- 总字数
- 识别的题材
- 叙事视角

### 5.2 实体清单

- 角色列表（含别名、类型、首次出场、最近出场）
- 地点列表
- 组织列表
- 物品/能力列表

### 5.3 结构分析

- 卷划分（如有）
- 主线描述
- 副线描述
- 情节节奏（高潮/低谷分布）
- 伏笔清单（已闭合/未闭合）

### 5.4 风格指纹

- 句式偏好（长短句比例、句式结构偏好）
- 词汇特征（高频词、回避词）
- 对话风格（口语化程度、潜台词使用）
- 叙事视角（人称、距离）
- 情绪描写模式（行为暗示 vs 直接标签）

### 5.5 冲突清单

- 设定冲突列表（含严重度和置信度）
- 时间线矛盾列表
- 未闭合伏笔列表

### 5.6 接管建议

- 建议从哪一章开始续写
- 需要修复的优先问题
- 风格续写的关键注意事项

## 六、降级策略

当某些分析无法完成时的降级处理：

| 分析项 | 失败原因 | 降级处理 |
|--------|----------|----------|
| 结构分析 | 文本太短（< 5 章） | 跳过卷结构，只做主线识别 |
| 风格指纹 | 文本质量太低 | 标记"风格指纹不可靠"，人工确认 |
| 冲突检测 | 设定信息不足 | 只检测明显矛盾，标记"需人工复核" |
| 伏笔识别 | 文本太散 | 标记"伏笔识别不完整"，人工补充 |
