# 长篇小说创作指南

## 结构模式

### 三幕式结构
- **第一幕（25%）**：建立日常世界 → 引入冲突 → 第一个转折点
- **第二幕（50%）**：冲突升级 → 盟友与敌人 → 中点反转 → 至暗时刻
- **第三幕（25%）**：最终对决 → 高潮 → 结局 → 新常态

### 英雄之旅（12阶段）
1. 平凡世界 → 2. 冒险召唤 → 3. 拒绝召唤 → 4. 导师相遇 → 5. 跨越门槛 → 6. 试炼盟友敌人 → 7. 接近最深洞穴 → 8. 磨难 → 9. 报酬 → 10. 归途 → 11. 复活 → 12. 携万灵药归来

### 多线叙事结构
- 主线 + 2-3条副线
- 副线与主线在关键节点交汇
- 视角切换要有意义（传递主线无法获知的信息）

## 章节设计

### 章节长度
- 通俗小说：3000-5000字/章
- 纯文学：不限，可长可短
- 类型小说：2000-4000字/章

### 章节节奏
- 每章应有"微型故事弧"：开端→发展→小高潮/悬念
- 章末钩子：提出问题、暗示转折、情感悬置
- 紧张与舒缓交替，避免连续3章以上的高强度

### 章节命名
- 序号式：第一章、第二章
- 标题式：概括本章核心意象或事件
- 双重式：序号+标题

## 视角选择

| 视角 | 优势 | 劣势 | 适用场景 |
|------|------|------|----------|
| 第一人称 | 亲密感、悬疑 | 信息受限 | 成长小说、悬疑 |
| 第三人称有限 | 灵活+聚焦 | 需控制信息量 | 大多数类型小说 |
| 第三人称全知 | 宏大叙事 | 容易散 | 史诗、群像 |
| 第二人称 | 沉浸感 | 难以驾驭 | 实验性作品 |

## 人物弧光设计

### 主角弧光类型
- **正向弧光**：从缺陷到成长（最常见）
- **负向弧光**：从光明到堕落（悲剧）
- **平坦弧光**：主角不变，改变周围世界（信念型英雄）

### 弧光设计模板
```
起点状态：主角相信什么错误观念？
触发事件：什么打破了日常？
试炼过程：哪些事件迫使主角面对自己的缺陷？
转折时刻：主角在什么时刻意识到真相？
终点状态：主角发生了怎样的变化？
```

## 小说类型要素

### 推理/悬疑
- 诡计设计：公平性（读者有足够线索推理）
- 线索管理：明线、暗线、误导线
- 叙述性诡计：利用视角/信息差制造反转

### 科幻
- 世界观自洽：科技规则一旦建立不可随意打破
- 思想实验：如果X技术/社会制度存在，会怎样？
- 硬科幻 vs 软科幻：技术细节 vs 人文思考

### 奇幻
- 魔法体系：有代价的力量更有趣
- 世界构建：政治、经济、文化、地理
- 种族设计：避免简单善恶二元

### 言情/爱情
- 情感节奏：相遇→吸引→障碍→突破→考验→结局
- 配对化学反应：互补或相似，但必须有张力
- 配角不工具化：每个配角都有自己的故事

## 常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 中盘疲软 | 缺乏中期目标 | 设置中点反转，改变游戏规则 |
| 人物脸谱化 | 缺乏内在矛盾 | 给每个角色一个秘密或矛盾 |
| 结局仓促 | 高潮后缺少收尾 | 高潮后留10%篇幅收束 |
| 节奏失控 | 描写/叙述比例失衡 | 对话推进剧情，描写控制情绪 |
