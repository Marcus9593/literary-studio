# 漫画/动画脚本指南

## 核心概念

漫画和动画脚本是"视觉叙事"——用画面讲故事。编剧需要：
- 用画面思维而非文字思维
- 考虑镜头语言（构图、角度、景别）
- 控制每页/每帧的信息量
- 理解翻页/转场的节奏感

## 漫画脚本格式

### 页面格式
```
第1页（6格）

格1 [全景 - 外景 - 城市街道 - 白天]
俯瞰视角。繁华的城市街道，人来人往。
右下角一个小身影在人群中穿梭。

格2 [中景 - 外景 - 街道]
跟拍小身影。是一个背着大书包的女孩，大约12岁。
她低头看手机，脚步匆忙。

格3 [近景 - 女孩的脸]
女孩突然停下，眼睛睁大。
女孩：什么……？

格4 [特写 - 手机屏幕]
屏幕上显示一条消息："检测到异常能量波动。
位置：你脚下。"

格5 [全景 - 街道]
女孩猛然抬头。周围一切正常，行人在正常行走。
但地面有一道细微的裂缝在发光。

格6 [特写 - 女孩的眼睛]
瞳孔中倒映出那道光。
女孩：（小声）不可能……
```

### 格式要点
- 每页标明格数
- 每格标明：景别、场景、时间
- 描述画面内容（画师能看到的）
- 对白和旁白单独标出
- 音效用拟声词

## 动画脚本格式

### 基本格式
```
场景1  外景  城市上空  白天

【大全景。镜头从云层穿过，俯冲向城市。
城市的全貌逐渐清晰——高楼林立，飞行器穿梭。
镜头跟随一辆红色飞行摩托，穿过建筑群。

驾驶者（画外音，女声）
今天的任务，代号"红蝶"。

摩托急速下降，冲向一座摩天大楼的天台。

切换：

场景2  外景  摩天大楼天台  白天

摩托稳稳降落。驾驶者摘下头盔——
是一个20多岁的红发女子，表情冷峻。

红蝶
（对通讯器）
已到达目标位置。
准备执行。

她从摩托上取下一个银色箱子，走向天台边缘。
俯瞰城市，嘴角微微上扬。

红蝶
（自言自语）
游戏开始。
```

## 视觉叙事技巧

### 镜头语言
| 景别 | 用途 | 情感效果 |
|------|------|----------|
| 大远景 | 建立环境 | 渺小、孤独、宏大 |
| 全景 | 展示人物与环境关系 | 上下文、处境 |
| 中景 | 展示动作和互动 | 叙事性 |
| 近景 | 展示表情和情绪 | 亲密、紧张 |
| 特写 | 强调细节 | 冲击力、重要性 |
| 大特写 | 极端强调 | 震撼、悬疑 |

### 角度与构图
- **平视**：客观、中性
- **俯视**：渺小、被压迫
- **仰视**：威严、威胁
- **倾斜**：不安、混乱
- **鸟瞰**：上帝视角、全局

### 翻页节奏（漫画特有）
- **翻页前**：设置悬念（读者翻页前看到的最后一个画面很重要）
- **翻页后**：给冲击力（翻开后第一个画面要有视觉冲击）
- **跨页大图**：重要时刻用跨页，制造震撼效果

## 对白与旁白

### 对白原则
- **气泡容量**：一个气泡不超过20-25个字（中文）
- **分格控制**：重要对白独占一格
- **节奏感**：对白和画面交替，不要一格塞太多文字
- **个性化**：每个角色的说话方式要不同

### 旁白/内心独白
- 用文本框（不同于对白气泡）
- 颜色/样式区分叙述者
- 控制使用频率，不要变成"带插图的小说"

### 音效文字
- 融入画面构图
- 大小和样式反映声音大小和质感
- 中文音效：砰、咔嚓、嗖、嗡——
- 日式风格：ドドド、ゴゴゴ（根据作品风格选择）

## 类型要点

### 少年漫画
- 节奏快，动作场面多
- 升级体系明确
- 友情、努力、胜利的主题
- 格子大，动作线条粗

### 少女漫画
- 情感细腻，表情描写多
- 画面精致，留白较多
- 内心独白丰富
- 格子变化多，节奏较慢

### 青年漫画
- 主题深刻，可探讨社会议题
- 画面写实，细节丰富
- 对白质量高，不回避复杂话题
- 节奏自由

### 搞笑漫画
- 节奏是生命（梗的时间点很重要）
- 夸张的表情和动作
- 吐槽和反差
- 格子不要太满，留出"笑点呼吸"的空间

## 系列规划

### 单话结构（连载漫画）
```
开头（前3-4页）：承接上话 / 建立本话情境
发展（中间10-12页）：核心事件
高潮（3-4页）：本话最精彩的画面
结尾（1-2页）：悬念钩子 / 下话预告
```
- 标准连载话：19-22页
- 每话应有独立的小高潮
- 同时推进主线

### 长期规划
- **大纲**：整个系列的主线走向
- **分卷规划**：每卷（约8-10话）有一个完整的冲突弧
- **角色日历**：每个角色的成长节点
- **伏笔管理**：记录埋下的伏笔和计划的回收时间
