# 特鲁比 22 步故事结构完整指南

> 基于约翰·特鲁比（John Truby）《故事的解剖》（*The Anatomy of Story*）提炼的 22 步故事结构。
> 本文件为编剧/小说家的深度参考，涵盖定义、对照、体裁变体与自检方法。

---

## 一、22 步总览

特鲁比将故事视为"主角在道德与心理层面的成长之旅"，每一步都推动主角从"虚假自我"走向"真实自我"。

| 步骤 | 英文名 | 中文名 | 核心功能 |
|------|--------|--------|---------|
| 1 | Self-Revelation, Need, Desire | 虚假自我与潜在需求 | 建立主角的心理缺陷与道德盲点 |
| 2 | Desire | 欲望 | 主角的外在目标浮出水面 |
| 3 | Opponent | 对手 | 与主角争夺同一目标的对立力量 |
| 4 | Plan | 计划 | 主角为达成目标采取的策略 |
| 5 | Battle | 对决 | 主角与对手的终极冲突 |
| 6 | Self-Revelation | 自我揭示 | 主角面对真相，认识真实的自我 |
| 7 | New Equilibrium | 新平衡 | 故事结束后的世界状态 |
| 8 | Ghost & Story World | 幽灵与故事世界 | 过去的创伤如何塑造当下的世界 |
| 9 | Weakness & Need | 缺陷与需求 | 主角的内在弱点与真正的需求 |
| 10 | Inciting Incident | 激励事件 | 打破平衡的外部事件 |
| 11 | Desire (revisited) | 欲望（深化） | 主角在激励事件后明确的目标 |
| 12 | Ally/Helper | 盟友/助手 | 陪伴主角、弥补主角缺陷的角色 |
| 13 | Opponent & Their Plan | 对手及其计划 | 对手的动机与策略 |
| 14 | Attack by Ally | 盟友的攻击 | 盟友推动主角采取行动 |
| 15 | Apparent Defeat | 表面失败 | 主角陷入最低谷 |
| 16 | Second Revelation | 第二次揭示 | 主角获得更深层次的认知 |
| 17 | Third Revelation & Decision | 第三次揭示与抉择 | 主角做出关键道德选择 |
| 18 | Gate / Dark Moment | 关隘/黑暗时刻 | 主角面对最大考验 |
| 19 | Battle (climax) | 对决（高潮） | 全剧的终极对抗 |
| 20 | Self-Revelation (protagonist) | 主角的自我揭示 | 主角完成内在蜕变 |
| 21 | Moral Decision | 道德抉择 | 主角以行动证明改变 |
| 22 | New Equilibrium | 新平衡 | 变化后的世界与主角 |

> **说明**：特鲁比体系有多种整理方式，不同版本的步骤编号与名称略有差异。本指南以功能为核心，将 22 步归纳为 7 个"主结构点" + 15 个"支撑节拍"，便于实际应用。

## 二、22 步详解（含体裁变体）

### 步骤 1：虚假自我 / 幽灵（False Self / Ghost）

**定义**：故事开始前，主角活在一个"虚假自我"中——由某个过去的创伤（幽灵）塑造的心理防御机制。主角的缺陷（Weakness）和表面欲望（Desire）都源自这个虚假自我。

**功能**：建立主角的起点状态，埋下成长的种子。

| 体裁 | 变体 |
|------|------|
| 网文 | 主角是"废材"——实力低微但内心不甘；幽灵常是童年创伤、家族灭门、天赋被夺 |
| 影视 | 主角表面成功但内心空虚——如事业有成的律师回避家庭问题 |
| 小说 | 主角活在自我欺骗中——如一个"幸福"的妻子不敢面对婚姻的真相 |

**自检**：读者/观众能否在开场 10 分钟（或前 3 章）内识别主角的缺陷？

---

### 步骤 2：欲望（Desire）

**定义**：主角想要什么？这是驱动整个故事的外在目标。欲望必须具体、可衡量、有时限。

**功能**：为故事提供引擎，让观众知道"赢"是什么样子。

| 体裁 | 变体 |
|------|------|
| 网文 | 复仇、变强、救亲人、登顶——欲望通常与"实力等级"挂钩 |
| 影视 | 赢得比赛、追到心上人、破案、逃脱——欲望可在一句话内说清 |
| 小说 | 可以更抽象：寻找归属感、理解死亡、重建身份 |

**自检**：能否用一句话说清主角想要什么？

---

### 步骤 3：对手（Opponent）

**定义**：与主角争夺同一目标的人（或力量）。对手不是"纯粹的恶"，而是"另一种答案"——他与主角面对同样的问题，但选择了不同的道路。

**功能**：制造冲突，迫使主角面对自己的缺陷。

| 体裁 | 变体 |
|------|------|
| 网文 | 对手往往是"天才型"或"特权型"——叶良辰、宗门大师兄、敌对势力少主 |
| 影视 | 对手与主角在价值观上对立——如《黑暗骑士》小丑 vs 蝙蝠侠 |
| 小说 | 对手可以是内在的——如社会制度、时间、主角自己的另一面 |

**自检**：对手是否有合理的动机？观众能否理解（甚至同情）对手？

---

### 步骤 4：计划（Plan）

**定义**：主角为达成欲望目标采取的策略。计划应该逐步展开——先用旧方法（失败），再用新方法（部分成功），最后用蜕变后的方法（最终胜利）。

**功能**：让故事有逻辑推进力，避免主角"运气好"。

| 体裁 | 变体 |
|------|------|
| 网文 | 计划 = 修炼路线 + 战斗策略——先硬拼（失败），再用智慧/外挂（成功） |
| 影视 | 计划有明确的步骤——如《十一罗汉》的分阶段执行 |
| 小说 | 计划可以是隐性的——主角以为自己在做 A，实际在做 B |

**自检**：主角的计划是否有逻辑？是否有"计划赶不上变化"的时刻？

---

### 步骤 5：对决（Battle）

**定义**：主角与对手的终极冲突。对决不仅是物理上的，更是价值观/道德层面的碰撞。

**功能**：故事的高潮，主角在压力下暴露真实自我。

| 体裁 | 变体 |
|------|------|
| 网文 | 对决 = 终极战斗——战力对比、大招释放、绝境翻盘 |
| 影视 | 对决 = 物理+心理双重战场——如法庭辩论+私下面对面 |
| 小说 | 对决可以是沉默的——如两个人在同一个房间里，做了一个决定 |

**自检**：对决是否同时解决了外部冲突和内部冲突？

---

### 步骤 6：自我揭示（Self-Revelation）

**定义**：主角在对决的压力下，终于看清了自己的真实面貌——承认缺陷，理解真正的需求（Need）。这是故事最深刻的时刻。

**功能**：完成主角的弧光，实现从虚假自我到真实自我的转变。

| 体裁 | 变体 |
|------|------|
| 网文 | 自我揭示常伴随"顿悟"——在生死关头领悟大道、理解武道真谛 |
| 影视 | 自我揭示通过选择展现——主角放弃欲望，选择需求 |
| 小说 | 自我揭示可以是反讽的——主角终于明白了，但代价巨大 |

**自检**：自我揭示是否来自主角自己的经历和选择，而非外力灌输？

---

### 步骤 7：新平衡（New Equilibrium）

**定义**：故事结束后的世界。主角已经改变（或证明无法改变），世界呈现新的秩序。

**功能**：给故事一个终点，让观众感受到变化的意义。

| 体裁 | 变体 |
|------|------|
| 网文 | 新平衡 = 飞升/称帝/归隐——留续集接口 |
| 影视 | 新平衡 = 与开场形成对照——开场的反面 |
| 小说 | 新平衡可以是开放的——变化已发生，但未来不确定 |

**自检**：新平衡是否与开场形成有意义的对照？

---

### 步骤 8：幽灵与故事世界（Ghost & Story World）

**定义**：幽灵是主角过去的创伤事件，它在故事开始前就已经发生，持续影响主角的行为。故事世界是主角当前生活的环境，它应该反映主角的内在状态。

**功能**：为主角的缺陷提供心理依据，让世界成为主角内心的外化。

| 体裁 | 变体 |
|------|------|
| 网文 | 幽灵 = 被灭门/被背叛/被夺舍；故事世界 = 修仙界的残酷法则 |
| 影视 | 幽灵 = 童年阴影/战场创伤/失去至亲；故事世界 = 反映主角心境的空间 |
| 小说 | 幽灵可以是社会性的——如阶级、种族、历史事件的余波 |

---

### 步骤 9：缺陷与需求（Weakness & Need）

**定义**：缺陷是主角的心理/道德弱点；需求是主角真正需要的东西（通常与欲望矛盾）。

**功能**：建立主角的成长空间。缺陷越大，成长越有戏剧性。

| 体裁 | 变体 |
|------|------|
| 网文 | 缺陷 = 鲁莽/自卑/贪婪/执念；需求 = 理解力量的真正含义 |
| 影视 | 缺陷 = 控制欲/逃避/骄傲/自私；需求 = 学会信任/面对/谦卑/付出 |
| 小说 | 缺陷可以更微妙——如过度理性、无法接受不完美 |

---

### 步骤 10：激励事件（Inciting Incident）

**定义**：打破主角日常生活平衡的外部事件。它给主角一个必须回应的问题，故事由此启动。

**功能**：点燃故事引擎。

| 体裁 | 变体 |
|------|------|
| 网文 | 退婚、亲人被害、获得系统、穿越重生 |
| 影视 | 发现尸体、收到录取通知、飞机失事、遇见真爱 |
| 小说 | 可以极微小——如一个电话、一封信、一个梦 |

---

### 步骤 11：欲望深化（Desire - Deepened）

**定义**：激励事件后，主角的欲望从模糊变具体，从被动变主动。主角开始追求一个明确的目标。

**功能**：让观众/读者清楚地知道故事的方向。

---

### 步骤 12：盟友/助手（Ally/Helper）

**定义**：陪伴主角、弥补主角缺陷的角色。盟友不是附庸，而是有自己目标和弧光的独立角色。

**功能**：提供情感支持、信息传递、互补能力，让主角不孤单。

| 体裁 | 变体 |
|------|------|
| 网文 | 兄弟/红颜/灵宠/系统——提供战斗辅助或情感支撑 |
| 影视 | 搭档/闺蜜/下属——有独立故事线，在关键时刻发挥作用 |
| 小说 | 盟友可以是反面的——如一个看似帮助主角但实际在利用主角的人 |

---

### 步骤 13：对手及其计划（Opponent & Their Plan）

**定义**：深入揭示对手的动机、计划和力量。对手也有自己的"幽灵"和"需求"。

**功能**：让对手立体化，增加冲突的深度。

---

### 步骤 14：盟友的攻击（Attack by Ally）

**定义**：盟友推动主角采取行动——可以是鼓励、质疑、甚至背叛。盟友不是"跟班"，而是有自己立场的角色。

**功能**：制造内部张力，迫使主角做出选择。

---

### 步骤 15：表面失败（Apparent Defeat）

**定义**：主角遭遇重大挫折，失去重要的东西——盟友、资源、信念。看似一切希望破灭。

**功能**：为最后的反弹蓄力。"最黑暗的时刻在黎明之前。"

| 体裁 | 变体 |
|------|------|
| 网文 | 师尊被杀、功力全废、被逐出宗门、众叛亲离 |
| 影视 | 被解雇、被抛弃、计划全面失败 |
| 小说 | 内在崩溃——信仰崩塌、身份认同危机 |

---

### 步骤 16：第二次揭示（Second Revelation）

**定义**：在表面失败的废墟中，主角获得第二个认知——比第一次更深。主角开始理解对手的世界观。

**功能**：为最终抉择做准备。

---

### 步骤 17：第三次揭示与抉择（Third Revelation & Decision）

**定义**：主角获得最终认知，并做出关键的道德选择——不是被迫的，而是主动的。

**功能**：主角的蜕变在行动中完成。

---

### 步骤 18：关隘/黑暗时刻（Gate / Dark Moment）

**定义**：主角面对最后的考验——跨越这道关隘，就进入对决。

**功能**：制造最大的紧张感。

---

### 步骤 19：对决/高潮（Battle - Climax）

**定义**：全剧的终极对抗。物理冲突与价值观冲突同时爆发。

**功能**：故事的最高点。

---

### 步骤 20：主角的自我揭示（Self-Revelation - Protagonist）

**定义**：对决中或对决后，主角完成最终的自我认知。

**功能**：弧光完成。

---

### 步骤 21：道德抉择（Moral Decision）

**定义**：主角以行动证明自己的改变——不是说"我变了"，而是"做了一个以前不会做的选择"。

**功能**：用行动而非语言证明成长。

---

### 步骤 22：新平衡（New Equilibrium）

**定义**：变化后的世界。与步骤 7 相同但更具体——主角已经不是从前的自己。

**功能**：故事的终点。

---

## 三、与三幕式结构对照表

| 三幕式 | 特鲁比 22 步 | 功能对应 |
|--------|-------------|---------|
| **第一幕：建置** | | |
| 开场 | 1-2（虚假自我、欲望） | 建立主角与世界 |
| 激励事件 | 10（激励事件） | 打破平衡 |
| 第一幕转折 | 12（盟友出现） | 主角开始行动 |
| **第二幕：对抗** | | |

## 四、与英雄之旅对照表

| 英雄之旅阶段 | 特鲁比 22 步 | 差异说明 |
|-------------|-------------|---------|
| 1. 平凡世界 | 1-2（虚假自我、欲望） | 特鲁比强调"虚假"而非"平凡" |
| 2. 冒险召唤 | 10（激励事件） | 相同功能 |
| 3. 拒绝召唤 | 9（缺陷与需求） | 特鲁比用"缺陷"解释拒绝的原因 |
| 4. 遇见导师 | 12（盟友/助手） | 特鲁比的"盟友"比"导师"更广 |
| 5. 跨越门槛 | 11（欲望深化） | 特鲁比强调"主动追求" |
| 6. 试炼与盟友 | 13-14（对手计划、盟友攻击） | 特鲁比更重视对手的主动性 |
| 7. 深入洞穴 | 15（表面失败） | 特鲁比的"失败"更强调道德层面 |
| 8. 苦难考验 | 16-18（二次揭示到关隘） | 特鲁比分三层揭示 |
| 9. 奖赏 | 17（第三次揭示与抉择） | 特鲁比的"奖赏"是认知而非物品 |
| 10. 返回之路 | 19（对决） | 特鲁比将"返回"融入对决 |
| 11. 复活 | 20（主角自我揭示） | 相同功能 |
| 12. 带着灵药回归 | 21-22（道德抉择、新平衡） | 特鲁比强调"道德行动"而非"万灵药" |

**关键差异**：
- 英雄之旅是"外在旅程"的模板，特鲁比是"内在蜕变"的蓝图
- 英雄之旅有"导师"角色，特鲁比用"盟友"替代（盟友有自己的弧光）
- 特鲁比特别强调"对手"的深度——对手不是工具人，而是"另一种答案"

## 五、22 步与网文分卷映射

### 每卷独立运行的"微型 22 步"

长篇网文每卷都可以视为一个独立的"微型故事"，拥有自己的 22 步结构：

| 微型步骤 | 对应全书步骤 | 卷内功能 | 章节占比建议 |
|---------|-------------|---------|-------------|
| 卷内虚假自我 | 全书步骤 1 的局部展示 | 本卷主角的初始状态 | 前 5% |
| 卷内欲望 | 全书步骤 2 的阶段性目标 | 本卷的具体目标 | 前 10% |
| 卷内对手 | 全书步骤 3 的阶段性对手 | 本卷的小 BOSS | 前 15% |
| 卷内计划 | 全书步骤 4 的阶段性策略 | 主角在本卷的行动路线 | 15-30% |
| 卷内盟友 | 全书步骤 12 的阶段性伙伴 | 本卷的新角色 | 20-40% |
| 卷内表面失败 | 全书步骤 15 的阶段性挫折 | 本卷的低谷 | 50-70% |
| 卷内对决 | 全书步骤 19 的阶段性战斗 | 本卷的高潮 | 80-90% |
| 卷内新平衡 | 全书步骤 22 的阶段性结果 | 本卷结尾 + 下卷伏笔 | 90-100% |

### 分卷映射示例（以修仙文为例）

```
全书主线：废材少年 → 修仙大能

第一卷：新手村（步骤 1-7 的局部）
  虚假自我：被退婚的废材
  欲望：变强，打脸前未婚妻
  对手：叶良辰（天才型）
  计划：获得系统，苦修
  表面失败：被逐出家族
  对决：宗门大比
  新平衡：成为宗门新星

第二卷：宗门风云（步骤 8-15 的局部）
  幽灵：发现父亲留下的秘密
  新欲望：找到妹妹
  新对手：宗门内的阴谋家
  表面失败：师尊被杀
  新平衡：获得关键线索

第三卷：救妹之旅（步骤 16-22 的局部）
  第二次揭示：妹妹的真正身份
  对决：与血神教主的终极战斗
  自我揭示：理解力量的真谛
  新平衡：飞升仙界
```

## 六、常见失误与自检清单

### 常见失误

| 失误 | 表现 | 后果 | 修正方法 |
|------|------|------|---------|
| 虚假自我太弱 | 主角没有明显缺陷 | 成长弧光不明显 | 给主角一个具体的、有后果的缺陷 |
| 欲望太模糊 | "他想变强"但没有具体标准 | 故事缺乏方向感 | 给欲望加上可衡量的指标和时限 |
| 对手工具化 | 对手只是"坏人"，没有自己的逻辑 | 冲突缺乏深度 | 为对手设计幽灵、需求和计划 |
| 计划太顺利 | 主角一路顺风 | 读者失去紧张感 | 每次行动都遭遇意外，迫使主角调整策略 |
| 表面失败太轻 | 主角失去的不够多 | 读者感受不到压力 | 让主角失去最珍视的东西（盟友/信念/能力） |
| 自我揭示靠外力 | 别人告诉主角真相 | 弧光不真实 | 主角必须通过自己的经历和选择理解真相 |
| 新平衡与开场无关 | 结局像另一个故事 | 感觉不完整 | 新平衡必须是开场的"回答" |
| 盟友太工具化 | 盟友只是提供帮助的NPC | 人物扁平 | 盟友有自己的目标、缺陷和弧光 |
| 揭示层次不够 | 只有一次揭示就完成蜕变 | 蜕变太突兀 | 至少三次揭示，层层递进 |
| 对决只有物理层面 | 打赢了就完了 | 主题不深刻 | 对决必须同时是价值观的碰撞 |

### 自检清单

#### 整体结构自检

- [ ] 主角是否有明确的"虚假自我"（缺陷+幽灵）？
- [ ] 主角的欲望是否具体、可衡量、有时限？
- [ ] 对手是否与主角争夺同一目标？
- [ ] 对手是否有合理的动机和自己的计划？
- [ ] 盟友是否有独立的弧光和目标？
- [ ] 主角的计划是否经历"旧方法失败→新方法部分成功→蜕变后方法胜利"的递进？
- [ ] 是否有至少一次"表面失败"？
- [ ] 是否有至少三次"揭示"（认知递进）？
- [ ] 对决是否同时解决外部冲突和内部冲突？
- [ ] 自我揭示是否来自主角自己的经历，而非外力灌输？
- [ ] 新平衡是否与开场形成对照？
- [ ] 主角的道德抉择是否通过行动而非语言证明？

#### 分卷自检（网文专用）

- [ ] 每卷是否有独立的"微型弧光"（从起点到高潮到收束）？
- [ ] 每卷的阶段性对手是否比上一卷更强？
- [ ] 每卷的表面失败是否比上一卷更严重？
- [ ] 卷与卷之间的伏笔是否有计划回收？
- [ ] 全书的三次揭示是否分布在不同的卷中？

#### 体裁适配自检

**网文**：
- [ ] 每 10-20 章是否有至少一个"爽点"或"悬念点"？
- [ ] 打脸场景是否利用了"对手低估主角"的认知落差？
- [ ] 换地图时是否重置了对手层级？
- [ ] 金手指/系统是否有代价和限制？

**影视**：
- [ ] Logline 能否一句话说清主角的欲望？
- [ ] 每一幕的转折是否在视觉上可表现？
- [ ] 对白是否有潜台词？
- [ ] 制作成本是否在合理范围内？

**小说**：
- [ ] 叙述视角是否一致（或切换有理由）？
- [ ] 内心描写是否推动了情节（而非纯抒情）？
- [ ] 文风是否与故事调性匹配？
