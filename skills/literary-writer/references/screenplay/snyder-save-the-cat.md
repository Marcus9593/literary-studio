# 斯奈德 10 种故事类型（Save the Cat）

> 基于布莱克·斯奈德（Blake Snyder）《救猫咪》（Save the Cat）系列著作。
> 斯奈德将所有故事归纳为 10 种基本类型，每种类型都有其内在的戏剧引擎和节拍规律。

---

## 一、10 种故事类型详解

### 1. Monster in the House（屋中怪物）

**核心公式**：怪物 + 一个封闭空间 + 人物的过错（sin）

**戏剧引擎**：恐惧——观众害怕怪物会抓住角色。

**关键要素**：
- **怪物**：可以是超自然生物、变异体、杀手、病毒，甚至是不可见的"规则"
- **封闭空间**：物理封闭（房子、飞船、孤岛）或社会封闭（小镇、公司、家庭）
- **过错**：角色（或社会）的某个错误导致了怪物的诞生/入侵，这是"罪有应得"

**类型变体**：
- 纯恐怖型：《异形》《电锯惊魂》
- 社会隐喻型：《釜山行》《生化危机》
- 悬疑型：《禁闭岛》《致命ID》

**中国网文映射**：
- 无限流小说（封闭副本空间 + 怪物 + 角色的"原罪"）
- 规则怪谈（封闭空间 + 不可名状的规则怪物）
- 末日求生（封闭据点 + 丧尸/变异体）

---

### 2. Golden Fleece（金羊毛）

**核心公式**：上路（Road）+ 团队 + 成长

**戏剧引擎**：旅程本身——观众关注的是"路上发生了什么"，而非终点。

**关键要素**：
- **道路/旅程**：物理旅程（公路片）或隐喻旅程（人生阶段）
- **团队**：主角不是一个人，旅程中需要同伴
- **自我发现**：真正被改变的不是外在目标，而是主角的内心

**类型变体**：
- 公路片：《人在囧途》《阳光小美女》
- 任务型：《指环王》《西游记》
- 成长型：《千与千寻》《阿甘正传》

**中国网文映射**：
- 修仙升级流（修炼之路 = 隐喻旅程，师门/队伍 = 团队）
- 西游同人/玄幻冒险（显性公路结构）
- 系统流（完成任务 = 走过不同"站点"）

---

### 3. Out of the Bottle（愿望成真）

**核心公式**：许愿 + 魔法实现 + 代价与教训

**戏剧引擎**：反讽——愿望实现了，但结果不是你想要的。

**关键要素**：
- **许愿**：主角明确表达一个愿望（或被赋予超能力）
- **魔法实现**：愿望成真，生活发生剧变
- **"小心你许的愿"**：愿望的实现揭示了主角真正需要的东西

**类型变体**：
- 身体互换：《你的名字》《变身辣妹》
- 超能力获得：《冒牌天神》《蜘蛛侠》
- 穿越/重生：回到过去改变命运

**中国网文映射**：
- 穿越重生（前世许愿/遗憾 → 今生实现 → 发现代价）
- 系统流（获得系统 = 愿望成真 → 系统的规则 = 代价）
- 都市异能（普通人获得超能力 → 发现能力的副作用）

---

### 4. Dude with a Problem（普通人遇大麻烦）

**核心公式**：普通人 + 突发危机 + 极端处境

**戏剧引擎**：同理心——观众代入"如果是我该怎么办"。

**关键要素**：
- **普通人**：主角没有特殊技能或资源，就是你我这样的普通人
- **突发危机**：一个极端事件打破了日常（被冤枉、被绑架、遭遇灾难）
- **以小博大**：主角必须用普通人的智慧和勇气对抗远超自己的力量

**类型变体**：
- 灾难型：《2012》《釜山行》
- 冤屈型：《亡命天涯》《肖申克的救赎》
- 生存型：《火星救援》《荒野猎人》

**中国网文映射**：
- 都市爽文（小人物被欺负 → 反击权贵）
- 末日求生（普通人面对世界级灾难）
- 宫斗（普通宫女卷入宫廷阴谋）

---

### 5. Rites of Passage（人生仪式）

**核心公式**：人生阶段转变 + 痛苦 + 自我认知

**戏剧引擎**：共鸣——观众在角色身上看到自己经历过的某个阶段。

**关键要素**：
- **人生节点**：青春期、中年危机、衰老、丧亲、成瘾、离婚……
- **痛苦**：转变过程必然伴随痛苦，没有捷径
- **自欺→自知**：主角必须放弃对自己的幻想，接受真实的自己

**类型变体**：
- 成长型：《少年派的奇幻漂流》《伯德小姐》
- 中年危机型：《美国丽人》《一次离别》
- 丧亲/失恋型：《海边的曼彻斯特》《前任3》

**中国网文映射**：
- 校园青春（高考、初恋、友情破裂）
- 都市情感（婚姻危机、职场迷茫）
- 仙侠中的"心魔劫"（内心成长的隐喻）

---

### 6. Buddy Love（伙伴情）

**核心公式**：不完整的两人 + 相遇 + 缺憾弥补

**戏剧引擎**：化学反应——两个不完整的人在一起产生了1+1>2的效果。

**关键要素**：
- **不完整**：每个角色都有对方没有的东西
- **催化剂事件**：让两个本不会在一起的人被迫相处
- **"如果没有你"**：结局揭示如果没有对方，主角会成为什么样的人

**类型变体**：
- 爱情型：《泰坦尼克号》《你的名字》
- 友情型：《触不可及》《绿皮书》
- 对手型：《蝙蝠侠：黑暗骑士》（蝙蝠侠与小丑）

**中国网文映射**：
- 甜宠文（高冷男主 × 活泼女主 = 互补）
- 耽美/百合（两个性格互补的角色）
- 对手文（宿敌之间的惺惺相惜）

---

### 7. Whydunit（为什么这样做）

**核心公式**：犯罪/谜团 + 深入调查 + 揭示人性黑暗面

**戏剧引擎**：好奇心——观众想知道的不是"谁做的"，而是"为什么"。

**关键要素**：
- **犯罪/谜团**：一个引人注目的犯罪或不可解释的事件
- **调查者**：一个执着的追寻者（侦探、记者、普通人）
- **人性深渊**：调查过程让调查者不得不面对人性的黑暗面，自己也被改变

**类型变体**：
- 侦探型：《七宗罪》《利刃出鞘》
- 心理型：《沉默的羔羊》《隐秘的角落》
- 社会型：《熔炉》《我不是药神》

**中国网文映射**：
- 悬疑推理（刑侦、法医视角的社会派推理）
- 宫斗权谋（为什么这个人会变成这样？）
- 黑暗系网文（反派的堕落之路 = whydunit 的变体）

---

### 8. The Fool Triumphant（傻子胜利）

**核心公式**：被低估的主角 + 体制/强权 + 意外胜利

**戏剧引擎**：满足感——观众看着"傻子"用真诚和善良打败自以为是的强者。

**关键要素**：
- **"傻子"**：被所有人低估的人（天真的、残疾的、社会底层的）
- **强大的对手/体制**：代表"聪明"和"权力"的存在
- **以弱胜强**：傻子的"傻"恰恰是他的武器（真诚、善良、不按常理出牌）

**类型变体**：
- 喜剧型：《阿甘正传》《憨豆先生》
- 励志型：《摔跤吧！爸爸》《中国女排》
- 讽刺型：《皇帝的新装》式的故事

**中国网文映射**：
- 废柴逆袭（被嘲笑的主角最终打脸所有人）
- 扮猪吃虎（装傻充愣，实际深藏不露）
- 小人物逆袭（底层人物通过努力或奇遇翻身）

---

### 9. Institutionalized（制度化）

**核心公式**：群体/体制 + 个人 vs 集体 + 选择

**戏剧引擎**：认同危机——观众思考"我是该融入还是该反抗"。

**关键要素**：
- **群体/体制**：一个强大的组织（公司、军队、黑帮、家族、社会规范）
- **个人选择**：融入（失去自我）、反抗（失去归属）、或找到第三条路
- **群体思维的代价**：展示体制如何吞噬个体

**类型变体**：
- 职场型：《穿普拉达的女王》《杜拉拉升职记》
- 帮派型：《教父》《无间道》
- 社会型：《楚门的世界》《V字仇杀队》

**中国网文映射**：
- 宫斗/宅斗（个人在家族/宫廷体制中的生存）
- 职场文（个人在公司体制中的选择）
- 修仙宗门（个人在宗门规矩中的挣扎）

---

### 10. Superhero（超级英雄）

**核心公式**：非凡能力 + 世俗困扰 + 使命的代价

**戏剧引擎**：同情——拥有超凡能力的人也有普通人的烦恼，能力越大，代价越大。

**关键要素**：
- **非凡能力/地位**：主角拥有超越常人的力量、天赋或地位
- **世俗困扰**：即使是超级英雄也有普通人的问题（感情、家庭、身份认同）
- **使命的代价**：承担非凡使命必须付出的代价（孤独、牺牲、被误解）

**类型变体**：
- 超英型：《蜘蛛侠》《蝙蝠侠》
- 天才型：《心灵捕手》《美丽心灵》
- 历史英雄型：《辛德勒的名单》《林肯》

**中国网文映射**：
- 无敌流（主角天下无敌，但有情感/道义上的困扰）
- 天才流（天赋异禀，但承受常人不知的压力）
- 修仙大佬（实力登顶，但面对天道/命运的代价）

## 二、10 种类型的 Beat Sheet 模板

> 每种类型遵循斯奈德 15 节拍结构（Beat Sheet），但各节拍的具体内容因类型而异。

### 通用 15 节拍框架

| 序号 | 节拍名称 | 位置 | 说明 |
|------|---------|------|------|
| 1 | Opening Image（开场画面） | 1% | 故事开始前的状态，与结尾形成对比 |
| 2 | Theme Stated（主题陈述） | 5% | 有人说出或暗示主题，主角尚未领悟 |
| 3 | Set-Up（铺垫） | 1-10% | 展示主角的日常世界、缺陷、身边的人 |
| 4 | Catalyst（催化剂） | 10% | 打破日常的事件，故事真正的起点 |
| 5 | Debate（争论） | 10-20% | 主角犹豫：要不要接受挑战？ |
| 6 | Break into Two（进入第二幕） | 20% | 主角做出选择，进入新世界 |
| 7 | B Story（B 故事线） | 22% | 通常是爱情线或友情线，承载主题 |
| 8 | Fun and Games（游戏时间） | 20-50% | 故事的"承诺兑现"，观众最期待的部分 |
| 9 | Midpoint（中点） | 50% | 虚假胜利或虚假失败，提高赌注 |
| 10 | Bad Guys Close In（坏人逼近） | 50-75% | 外部压力增大，内部团队裂痕出现 |
| 11 | All Is Lost（一切皆失） | 75% | 最低谷，可能有"死亡气息" |
| 12 | Dark Night of the Soul（至暗时刻） | 75-80% | 主角在绝望中反思 |
| 13 | Break into Three（进入第三幕） | 80% | 通过 B 故事线的领悟找到解决方案 |
| 14 | Finale（终局） | 80-99% | 运用新认知，打败对手，创造新世界 |
| 15 | Final Image（结尾画面） | 100% | 与开场画面对比，展示主角的改变 |

---

### 各类型 Beat Sheet 差异化要点

#### Monster in the House Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示封闭空间和角色的"过错"（sin） |
| Catalyst | 怪物首次出现或被发现 |
| Fun and Games | 逃生/对抗的"猫鼠游戏"，展示怪物的规则 |
| Midpoint | 角色以为找到了怪物的弱点（虚假胜利） |
| Bad Guys Close In | 怪物反击，同伴开始死亡 |
| All Is Lost | 怪物似乎不可战胜，空间被完全封锁 |
| Finale | 角色必须付出代价（承认过错）才能战胜怪物 |

#### Golden Fleece Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示主角的日常世界和不满 |
| Catalyst | 出发的理由（任务、逃亡、追寻） |
| Fun and Games | 旅途中的站点、遭遇、团队互动 |
| Midpoint | 旅途过半，主角开始改变（但可能不自知） |
| All Is Lost | 任务似乎失败，团队面临分裂 |
| Finale | 完成旅程，但真正的收获是内心的成长 |
| Final Image | 主角回到原点，但已经完全不同 |

#### Out of the Bottle Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示主角对现状的不满和愿望 |
| Catalyst | 愿望被实现（魔法降临） |
| Fun and Games | 享受愿望带来的好处（爽感） |
| Midpoint | 代价开始显现 |
| Dark Night of the Soul | 主角意识到"这不是我真正想要的" |
| Finale | 放弃或修正愿望，回到现实 |

#### Dude with a Problem Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 主角的普通人身份和日常 |
| Catalyst | 突发危机打破一切 |
| Fun and Games | 主角用普通人的智慧应对危机 |
| Midpoint | 主角暂时掌控局面（虚假胜利） |
| All Is Lost | 敌人远比想象中强大，所有资源耗尽 |
| Finale | 主角用最朴素的方式（而非超能力）解决问题 |

#### Rites of Passage Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示主角在人生节点前的状态 |
| Catalyst | 触发转变的事件（亲人离世、疾病、分手） |
| Debate | 主角试图逃避转变 |
| Fun and Games | 主角用各种方式逃避痛苦（但不奏效） |
| All Is Lost | 逃避带来的后果比面对更严重 |
| Dark Night of the Soul | 主角终于接受现实，直面痛苦 |
| Finale | 主角完成转变，获得新的自我认知 |

#### Buddy Love Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示两个主角各自的"不完整" |
| Catalyst | 两人相遇（通常是冲突性的相遇） |
| Fun and Games | 两人互动产生的化学反应 |
| Midpoint | 关系加深，但各自的缺陷开始显现 |
| Bad Guys Close In | 外部压力或内部误解导致分裂 |
| All Is Lost | 两人分开，各自回到不完整的状态 |
| Finale | 两人意识到彼此的重要性，重新在一起 |

#### Whydunit Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示调查者的执着和犯罪/谜团的发生 |
| Fun and Games | 调查过程，发现线索，接近真相 |
| Midpoint | 发现一个意想不到的线索，改变调查方向 |
| Bad Guys Close In | 调查者发现自己也被卷入其中 |
| All Is Lost | 真相的黑暗程度超出预期 |
| Finale | 揭示"为什么"，调查者被真相改变 |

#### The Fool Triumphant Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示"傻子"被低估和嘲笑的处境 |
| Catalyst | "傻子"被卷入一个远超其能力的情境 |
| Fun and Games | "傻子"用"傻"的方式意外成功 |
| Midpoint | "傻子"开始被认真对待（但对手仍轻视） |
| Bad Guys Close In | 对手开始认真对付"傻子" |
| All Is Lost | "傻子"的"傻"似乎不再管用 |
| Finale | "傻子"用最真诚的方式赢得最终胜利 |

#### Institutionalized Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示主角初入体制的新鲜/不安 |
| Catalyst | 体制对主角提出要求（融入或离开） |
| Fun and Games | 主角在体制中的适应/挣扎 |
| Midpoint | 主角开始享受体制的好处（但开始失去自我） |
| Bad Guys Close in | 体制的黑暗面暴露 |
| All Is Lost | 主角必须在体制和自我之间做出选择 |
| Finale | 主角做出选择：融入、反抗、或找到第三条路 |

#### Superhero Beat Sheet

| 节拍 | 类型特色 |
|------|---------|
| Set-Up | 展示主角的非凡能力和世俗困扰 |
| Catalyst | 一个只有主角能解决的危机出现 |
| Fun and Games | 主角用能力解决问题（爽感） |
| Midpoint | 主角发现能力的代价 |
| All Is Lost | 代价变得不可承受（失去重要的人/事物） |
| Dark Night of the Soul | 主角质疑自己的使命 |
| Finale | 主角接受代价，完成使命 |

## 三、10 种类型与中国网文/影视映射表

| 斯奈德类型 | 中国网文常见标签 | 中国影视代表作 | 核心卖点 |
|-----------|----------------|--------------|---------|
| Monster in the House | 无限流、规则怪谈、末日求生、副本 | 《开端》《隐秘的角落》 | 恐惧+求生+规则解密 |
| Golden Fleece | 修仙升级、玄幻冒险、系统流、公路冒险 | 《西游记》《流浪地球》 | 旅程+成长+团队 |
| Out of the Bottle | 穿越重生、系统流、都市异能、变身 | 《庆余年》《赘婿》 | 反差+爽感+代价 |
| Dude with a Problem | 都市爽文、宫斗、末日求生、逆袭 | 《我不是药神》《釜山行》 | 共情+以弱胜强 |
| Rites of Passage | 校园青春、都市情感、仙侠心魔 | 《你好，李焕英》《少年的你》 | 共鸣+成长+治愈 |
| Buddy Love | 甜宠、耽美、百合、对手文 | 《陈情令》《苍兰诀》 | 化学反应+互补+羁绊 |

### 4.2 基于结构的节拍检测

通过分析故事事件序列，检测是否符合 15 节拍结构：

```python
def detect_beat_sheet(events: list[dict]) -> dict:
    """
    基于故事事件列表，检测 15 节拍位置。

    检测逻辑：
    1. Opening Image: 第一个事件的状态描述
    2. Catalyst: 第一个 "value_shift" 或重大状态变化事件
    3. Break into Two: 主角进入新世界/接受任务的事件
    4. Midpoint: 总事件数 50% 位置的重大事件
    5. All Is Lost: 75% 位置的负面 value_shift
    6. Finale: 最后 20% 的高密度事件序列
    """
    total = len(events)
    if total < 5:
        return {"error": "事件数量不足，无法检测节拍"}

    beats = {}
    beats["opening_image"] = events[0]
    beats["final_image"] = events[-1]

    # Catalyst: 第一个 chapter >= 2 的重大状态变化
    for e in events:
        if e.get("chapter", 0) >= 2 and e.get("event_type") in (
            "value_shift_occurred", "character_state_changed"
        ):
            beats["catalyst"] = e
            break

    # Midpoint: 50% 位置
    mid_idx = total // 2
    beats["midpoint"] = events[mid_idx]

    # All Is Lost: 75% 位置的负面事件
    loss_idx = int(total * 0.75)
    beats["all_is_lost"] = events[loss_idx]

    return beats
```
