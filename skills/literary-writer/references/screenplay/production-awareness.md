# 制作可拍性与行业认知

## 一、编剧的制作意识

### 1.1 为什么要懂制作

编剧不是在真空中写作。你的剧本最终要变成影像，这意味着：
- 有人要出钱拍（预算）
- 有人要演（演员）
- 有人要拍（导演/摄影）
- 有人要剪（后期）
- 有人要审（平台/审查）

**懂制作的编剧 = 能写出可拍的剧本的编剧 = 被制片人喜欢的编剧。**

### 1.2 不懂制作的代价

| 问题 | 后果 |
|------|------|
| 场景太多 | 预算爆、拍摄周期拉长 |
| 外景夜景太多 | 成本翻倍 |
| 群演太多 | 协调困难、成本高 |
| 特效太多 | 后期成本不可控 |
| 角色太多 | 选角困难、演员档期冲突 |
| 季节限定 | 拍摄窗口窄 |

## 二、预算分级与编剧对策

### 2.1 场景成本分级

| 级别 | 场景类型 | 成本范围 | 编剧对策 |
|------|---------|---------|---------|
| S | 内景·摄影棚 | 最低 | 多用，可反复利用 |
| A | 内景·实景 | 低-中 | 可用，注意场景数量 |
| B | 外景·城市·日 | 中 | 控制数量 |
| C | 外景·城市·夜 | 中-高 | 少用 |
| D | 外景·自然 | 高 | 精打细算 |
| E | 外景·特殊 | 极高 | 谨慎使用 |
| F | 动作/特效/群演 | 极高 | 必须有充分理由 |

### 2.2 预算友好的写作技巧

| 技巧 | 说明 |
|------|------|
| 场景合并 | 把多个功能相近的场景合并到同一地点 |
| 内景优先 | 同样的戏，能写内景就别写外景 |
| 日景优先 | 同样的戏，能写日就别写夜 |
| 群演控制 | 能用5个人就别用50个 |
| 对话驱动 | 对话戏比动作戏便宜得多 |
| 固定场景 | 主场景越多越省钱（如《老友记》的咖啡馆） |
| 重复利用 | 让角色回到之前去过的地方 |

### 2.3 场景数量参考

| 剧本类型 | 场景数量参考 | 说明 |
|---------|-------------|------|
| 电影（90-120页） | 40-70场 | 低于40场可能节奏慢，高于70场可能太碎 |
| 电视剧单集（42页） | 20-35场 | |
| 微短剧单集（2-3页） | 2-5场 | |

## 三、演员与角色设计

### 3.1 选角友好的角色设计

| 设计要素 | 选角影响 | 编剧对策 |
|---------|---------|---------|
| 年龄范围 | 决定演员池大小 | 写"30多岁"而非"34岁" |
| 外貌描写 | 限制选角 | 给特征暗示而非具体描写 |
| 口音/方言 | 需要演员会说 | 考虑备选方案 |
| 特殊技能 | 需要演员会或用替身 | 用剪辑/镜头语言替代 |
| 戏份集中度 | 演员档期 | 主角戏份尽量集中拍摄 |

### 3.2 角色数量控制

| 剧本类型 | 核心角色 | 重要配角 | 龙套 |
|---------|---------|---------|------|
| 电影 | 2-4 | 3-5 | 按需 |
| 电视剧（单季） | 4-6 | 6-10 | 按需 |
| 微短剧 | 2-3 | 2-4 | 极少 |

### 3.3 群戏的拍摄挑战

- 群戏意味着更多演员同时在场 → 档期协调困难
- 群戏的对白更难写 → 每个人都要有声音
- 群戏的调度更复杂 → 减少不必要的物理动作

## 四、拍摄天数与剧本长度

### 4.1 中国影视行业拍摄周期参考

| 类型 | 集数 | 拍摄天数 | 备注 |
|------|------|---------|------|
| 院线电影 | - | 60-120天 | |
| 网络电影 | - | 20-40天 | |
| 长剧集 | 40集 | 120-180天 | |
| 中剧集 | 24集 | 80-120天 | |
| 短剧集 | 12集 | 40-60天 | |
| 微短剧 | 100集 | 15-30天 | 极快节奏 |

### 4.2 编剧对拍摄周期的意识

- **场景转换需要时间**——每换一个场景，剧组需要转场、布景、打光
- **外景受天气影响**——写"晴天"结果拍摄当天下雨，整个计划要调
- **夜景拍摄效率低**——灯光布置复杂，拍摄时间有限
- **动作戏耗时**——一场5分钟的动作戏可能需要拍3-5天

## 五、审查与平台要求

### 5.1 中国影视审查要点

| 题材 | 红线 | 安全做法 |
|------|------|---------|
| 涉案 | 不能详细展示犯罪手法 | 重在推理和人性 |
| 历史 | 重大事件不能戏说 | 大事不虚，小事不拘 |
| 鬼怪 | 不能有真鬼 | 科学/心理/梦境解释 |
| 暴力 | 不能过度血腥 | 暗示比展示更安全 |
| 情感 | 不能过度露骨 | 暧昧比直接更有张力 |
| 政治 | 不能涉及敏感话题 | 避开或架空 |
| 宗教 | 不能歪曲宗教 | 客观呈现 |

### 5.2 平台审查差异

| 平台 | 审查特点 | 编剧注意 |
|------|---------|---------|
| 爱奇艺 | 品质导向，尺度相对宽松 | 悬疑/犯罪可以更暗黑 |
| 优酷 | 商业导向 | 类型化要强 |
| 腾讯 | 全品类 | 注意IP改编的还原度 |
| 芒果 | 青春/甜宠 | 情感尺度注意 |
| 抖音/快手 | 微短剧审核快 | 但违规下架也快 |

### 5.3 安全写法

- 敏感题材用"架空"——不写真实地名、真实组织
- 暴力用"后果"代替"过程"——不写怎么打，写打完后的样子
- 情感用"暗示"代替"直接"——一个眼神比一句告白更有力
- 犯罪用"人性"包装——重点是为什么犯罪，不是怎么犯罪

## 六、合拍与跨媒介

### 6.1 中外合拍注意事项

| 维度 | 注意 |
|------|------|
| 文化差异 | 避免文化刻板印象 |
| 审查差异 | 中国审查标准与海外不同 |
| 格式差异 | 剧本格式需要双版本 |
| 语言差异 | 对白需要考虑翻译可行性 |
| 制作差异 | 工作流程和习惯不同 |

### 6.2 IP改编

| IP类型 | 改编挑战 | 编剧对策 |
|--------|---------|---------|
| 小说 | 篇幅压缩 | 选择核心线索，删支线 |
| 漫画 | 视觉化叙事转文字 | 保留视觉风格，补充内心 |
| 游戏 | 互动叙事转线性 | 提取世界观和角色，重构故事 |
| 真实事件 | 戏剧化与忠实的平衡 | 大事不虚，小事不拘 |
| 综艺 | 娱乐性与故事性的平衡 | 保留核心梗，补充角色深度 |

### 6.3 系列化开发

当你的项目有可能做成系列时：
- 第一季要完整——不能依赖"第二季会解释"
- 留钩子但不留硬伤——观众能接受悬念，不能接受逻辑漏洞
- 角色弧光要有阶段性终点——每季角色要有可感知的成长
- 世界观要可扩展——但第一季不要过度铺设

## 七、编剧的行业生存

### 7.1 交付物清单

| 阶段 | 交付物 | 格式要求 |
|------|--------|---------|
| 提案 | Logline + One-Pager | 简洁有力 |
| 开发 | Treatment + Beat Sheet | 散文体，可读性强 |
| 大纲 | 分集大纲 + 分场大纲 | 结构清晰 |
| 初稿 | 完整剧本 | 格式规范 |
| 修改 | 修改稿 + 修改说明 | 标注修改点 |
| 定稿 | 最终剧本 | 场次号锁定 |

### 7.2 剧本格式规范要点

```
字体：宋体/等线（中文）、Courier（英文）
字号：小四或12号
行距：1.5倍
页边距：上下2.54cm，左右3.17cm
场景标题：加粗
角色名：加粗居中
对白：缩进
动作段：正常段落
```

### 7.3 稿酬参考（2024-2026年中国市场）

| 类型 | 初级编剧 | 中级编剧 | 资深编剧 |
|------|---------|---------|---------|
| 电影剧本 | 10-30万 | 30-100万 | 100万+ |
| 电视剧（每集） | 1-3万 | 3-8万 | 8-20万+ |
| 微短剧（每集） | 2000-5000 | 5000-1万 | 1-3万 |
| 网络电影 | 5-15万 | 15-50万 | 50万+ |

注：以上为参考范围，实际取决于项目规模、平台、编剧咖位。

### 7.4 编剧合同关键条款

| 条款 | 说明 |
|------|------|
| 署名权 | 署名位置、署名方式 |
| 稿酬 | 总额、分期支付方式 |
| 修改次数 | 合同约定的免费修改次数 |
| 著作权 | 归属（通常归制片方） |
| 续集权 | 是否包含续集/衍生的权利 |
| 竞业限制 | 是否限制同类型创作 |
