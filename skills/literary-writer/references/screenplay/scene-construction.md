# 场景建构方法论

## 一、场景的本质

场景不是"发生在一个地方的事"，是**一次状态转换**。场景开始时世界是一个样子，结束时世界是另一个样子——如果没变，这场戏就没有存在的理由。

### 1.1 场景的最小单位

```
一个场景 = 一个地点 + 一段时间 + 至少一次状态转换
```

- **地点变了** → 新场景
- **时间跳了** → 新场景
- **人物进出改变了力量动态** → 新场景内的新节拍

### 1.2 场景的戏剧功能

每场戏必须完成至少一项：

| 功能 | 说明 | 检验标准 |
|------|------|---------|
| 推进剧情 | 发生了一件改变局面的事 | 删掉这场戏，后面的情节会断裂 |
| 揭示人物 | 通过选择/反应展示角色本质 | 角色做了一个观众没预料到的选择 |
| 传递信息 | 观众需要知道的关键信息 | 信息通过冲突或行动传递，不是播报 |
| 制造转折 | 好→坏，或坏→好 | 场景结束时局面与开始时相反 |
| 建立关系 | 角色之间的力量动态变化 | 一方从强势变弱势，或关系升温/降温 |

## 二、场景节拍设计

### 2.1 什么是节拍

节拍（Beat）是场景内部的一次情绪或力量的微转换。一场戏通常有 3-7 个节拍。

### 2.2 节拍设计模板

```
场景：[地点] - [时间]

节拍1：[初始状态] → [微变化]
  └─ 动作/对白：___
  └─ 力量持有者：角色A

节拍2：[新状态] → [微变化]
  └─ 动作/对白：___
  └─ 力量持有者：角色B

节拍3：[新状态] → [微变化]
  └─ 动作/对白：___
  └─ 力量持有者：角色A

...

终节拍：[最终状态]
  └─ 与开场状态的关系：相同/相反/升级
```

### 2.3 节拍示例

```
场景：审讯室 - 夜

节拍1：警察主导，嫌疑人沉默
  └─ 警察把文件拍在桌上："说吧。"
  └─ 力量：警察

节拍2：嫌疑人开始说话，但说的是假话
  └─ 嫌疑人："那天晚上我在家。"
  └─ 力量：嫌疑人（在控制信息）

节拍3：警察拿出证据，假话被戳穿
  └─ 警察把监控截图推过去："这是你？"
  └─ 力量：警察

节拍4：嫌疑人反问，暴露警察的弱点
  └─ 嫌疑人："你调取监控的手续合法吗？"
  └─ 力量：嫌疑人

节拍5：警察沉默——这是个问题
  └─ 力量：僵持

节拍6：警察换策略，用情感突破
  └─ 警察："你女儿在等你回家。"
  └─ 力量：警察（但方式变了）

终节拍：嫌疑人崩溃，开始说真话
  └─ 与开场的关系：力量反转——警察用弱点（情感）而非优势（权力）获胜
```

## 三、场景开头与结尾

### 3.1 场景开头：Late In, Early Out

**晚进早出**——在最接近冲突的时刻进入场景，在状态转换完成后立刻离开。

```
❌ 错误写法：
INT. 办公室 - 日
李明走进办公室，放下公文包，倒了一杯咖啡，坐下来打开电脑。
他看了一眼邮件，发现了一封来自老板的信。
他打开邮件，脸色变了。

✅ 正确写法：
INT. 办公室 - 日
李明盯着屏幕。邮件标题："关于你的去留问题——"
他的咖啡杯停在嘴边，忘了喝。
```

### 3.2 场景开头技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 中间开始 | 从冲突进行中开始 | 两人已经在争吵，观众慢慢明白原因 |
| 悬念开头 | 一个不寻常的画面 | 一个空椅子上放着一封信 |
| 感官切入 | 从声音/气味/触感开始 | 先听到哭声，再看到人 |
| 动作开头 | 角色正在做某事 | 一个人在埋东西 |
| 对比开头 | 与上一场形成强烈反差 | 上一场婚礼，这一场葬礼 |

### 3.3 场景结尾技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 悬念收尾 | 留一个问题未回答 | 电话响了，角色接起来，脸色变了。切。 |
| 意象收尾 | 一个视觉画面定格 | 杯子里的水满了，溢出来。 |
| 对白收尾 | 一句有力的话 | "你从来没有爱过我。" |
| 动作收尾 | 一个决定性的动作 | 他把戒指放在桌上，转身离开。 |
| 沉默收尾 | 角色不说话 | 她看着他的背影消失在走廊尽头。 |
| 声音收尾 | 声音延续到下一场 | 门关上的声音 → 切到教堂钟声 |

### 3.4 避免的结尾

- ❌ "然后他们就回去了。"——机械收尾
- ❌ 角色总结场景意义——"所以这件事告诉我们……"
- ❌ 无关的闲聊淡出——没有张力
- ❌ 镜头描述——"镜头缓缓拉远"（这是导演的事）

## 四、场景转换

### 4.1 对比转换

```
场景A结尾：角色笑着庆祝
场景B开头：另一个角色在哭泣

效果：情绪对比，暗示命运的不可预测
```

### 4.2 呼应转换

```
场景A结尾：角色说"我绝不会像我爸一样"
场景B开头：角色的爸在做同样的事

效果：讽刺，暗示命运的重复
```

### 4.3 声音桥接

```
场景A结尾：角色说"明天会更好"
场景B开头：（画外音）"明天"——另一个角色在完全不同的语境中说这个词

效果：时间/空间跳跃但有情绪连接
```

### 4.4 匹配剪辑

```
场景A结尾：角色关上一扇门
场景B开头：另一个角色打开一扇门

视觉相似，但含义完全不同
```

### 4.5 时间跳跃

```
场景A结尾：角色说"三个月后见"
场景B开头：字幕"三个月后"，直接进入新状态

注意：不要用旁白解释时间跳跃，用画面和状态告诉观众
```

## 五、场景的经济性

### 5.1 合并场景

如果两个场景：
- 地点相同或相近
- 时间连续
- 功能相似

→ 合并为一个场景。

```
❌ 三场戏：
1. 李明在办公室接到电话（信息）
2. 李明在走廊遇到张伟（关系）
3. 李明在会议室宣布消息（转折）

✅ 一场戏：
李明在会议室准备宣布消息，张伟进来，电话响了。
三个功能在一场戏里完成。
```

### 5.2 场景成本分级

| 级别 | 场景类型 | 拍摄难度 | 编剧对策 |
|------|---------|---------|---------|
| A级 | 内景·固定场地 | 最低 | 多用 |
| B级 | 内景·实景 | 低 | 可用 |
| C级 | 外景·城市 | 中 | 控制数量 |
| D级 | 外景·自然/夜景 | 高 | 少用 |
| E级 | 动作/特效/群演 | 极高 | 精打细算 |

### 5.3 场景清单模板

写完剧本后，列出所有场景：

```
# 场景清单

| # | 地点 | 内/外 | 时间 | 人物 | 功能 | 成本级 |
|---|------|------|------|------|------|--------|
| 1 | 咖啡馆 | 内 | 日 | 李明 | 建置 | B |
| 2 | 街道 | 外 | 夜 | 李明、张伟 | 关系 | C |
| ... | | | | | | |

总场景数：__
内景占比：__%
外景占比：__%
日景占比：__%
夜景占比：__%
```

## 六、场景诊断

### 6.1 场景问题检查表

| 问题 | 表现 | 解决 |
|------|------|------|
| 无功能 | 删掉不影响故事 | 删掉或合并到其他场景 |
| 节拍太平 | 力量始终在一方 | 加一个反转或抵抗 |
| 开头太慢 | 前半段无冲突 | 晚进——从冲突处开始 |
| 结尾太弱 | 没有状态转换 | 找到终节拍的转折点 |
| 信息播报 | 角色在解释背景 | 用冲突/行动传递信息 |
| 人物无选择 | 角色被动接受 | 给角色一个两难选择 |
| 时长失控 | 超过5页 | 拆分或删减节拍 |

### 6.2 场景增强技巧

| 技巧 | 说明 |
|------|------|
| 加限制 | 给场景一个时间/空间/规则限制（"你有5分钟"） |
| 加观众 | 让其他角色在场观看（增加压力） |
| 加秘密 | 至少一个角色有不能说的事 |
| 加赌注 | 提高这场戏的后果（赢了得什么，输了失什么） |
| 加意外 | 在节拍中插入预料之外的元素 |
| 加物理性 | 让角色在做某件具体的事（不只是站着说话） |
