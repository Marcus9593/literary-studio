# 类型片专项技法

## 一、悬疑推理

### 1.1 核心引擎
观众的驱动力是**"真相是什么"**——编剧必须管理信息的释放节奏。

### 1.2 信息管理矩阵

| 信息类型 | 观众知道 | 角色知道 | 效果 |
|---------|---------|---------|------|
| 双知 | ✅ | ✅ | 推进 |
| 观众独知 | ✅ | ❌ | 戏剧反讽/紧张 |
| 角色独知 | ❌ | ✅ | 悬念/好奇 |
| 双不知 | ❌ | ❌ | 探索/恐惧 |

悬疑片的核心技巧是**在四种信息状态之间切换**。

### 1.3 线索设计原则

- **公平原则**：观众和侦探看到同样的线索，理论上能自己推理出来
- **伪装原则**：关键线索要藏在大量信息中，不突出但不隐藏
- **误导原则**：设置"红鲱鱼"（假线索），让观众的推理跑偏
- **层叠原则**：真相不是一次性揭露，而是层层剥离

### 1.4 反转设计

| 反转类型 | 说明 | 示例 |
|---------|------|------|
| 身份反转 | 某人不是他/她自称的人 | 侦探就是凶手 |
| 动机反转 | 犯罪原因出乎意料 | 不是为钱，是为爱 |
| 时间反转 | 事件顺序打乱 | 《记忆碎片》 |
| 视角反转 | 叙事者不可靠 | 《第六感》 |
| 受害者反转 | "受害者"才是真凶 | 《消失的爱人》 |

### 1.5 经典结构模型

```
发现犯罪 → 侦探介入 → 调查（线索+误导）→ 误判 → 新线索 → 真相揭露
                                                    ↓
                                              中间至少一次假结论
```

## 二、喜剧

### 2.1 喜剧的本质

喜剧 = **预期的打破**。观众期待A，你给B，而且B比A更荒谬/更聪明/更真实。

### 2.2 喜剧类型

| 类型 | 驱动力 | 技法 | 代表作 |
|------|--------|------|--------|
| 情境喜剧 | 荒谬处境 | 规则建立+打破 | 《老友记》 |
| 讽刺喜剧 | 社会批判 | 夸张+反讽 | 《是，大臣》 |
| 浪漫喜剧 | 关系误会 | 信息差+巧合 | 《当哈利遇到萨莉》 |
| 黑色喜剧 | 禁忌话题 | 平淡语气+极端内容 | 《冰血暴》 |
| 闹剧 | 物理混乱 | 速度+节奏 | 《白头神探》 |
| 伪纪录片 | 真实感 | 假正经+真荒谬 | 《办公室》 |

### 2.3 喜剧节拍

```
铺垫（Setup）：建立预期
  ↓
延迟（Beat）：1-2秒的停顿（让观众形成预期）
  ↓
包袱（Punchline）：打破预期
  ↓
Tag（可选）：追加一个额外笑点（"还有更荒谬的"）
```

### 2.4 喜剧禁忌

- ❌ 解释笑话——如果需要解释，笑话就死了
- ❌ 角色知道自己在搞笑——最搞笑的角色往往最认真
- ❌ 用低俗代替聪明——可以有低俗，但不能只有低俗
- ❌ 连续同类型笑点——要变化（语言→情境→肢体→反转）

### 2.5 中国喜剧市场特点

- 观众偏好：情境荒谬 > 语言幽默 > 肢体搞笑
- 热门模式：身份错位、穿越反差、小人物逆袭
- 平台偏好：短视频喜剧（1-3分钟）需求大
- 注意：避免低俗化审查

## 三、恐怖惊悚

### 3.1 恐怖 vs 惊悚

| 维度 | 恐怖（Horror） | 惊悚（Thriller） |
|------|---------------|-----------------|
| 核心恐惧 | 未知/超自然/死亡 | 人性/阴谋/失控 |
| 观众状态 | 害怕 | 紧张 |
| 节奏 | 慢→爆发 | 持续高压 |
| 解决方式 | 通常无法彻底解决 | 通常可以解决 |

### 3.2 悬念工程

希区柯克的炸弹理论：
```
两个人在聊天，桌子底下有一颗炸弹。

方式A（惊吓）：炸弹突然爆炸。观众惊叫5秒。
方式B（悬念）：观众知道有炸弹，但角色不知道。
  观众紧张10分钟——他们聊什么不重要，观众只想着"快走！"
  炸弹可能爆炸，也可能被发现——这叫悬念。
```

**原则**：悬念 > 惊吓。让观众知道危险存在，比突然吓他们更有效。

### 3.3 恐怖技法

| 技法 | 说明 | 示例 |
|------|------|------|
| 限制视野 | 观众只能看到一部分 | 手电筒照亮的范围 |
| 声音先行 | 先听到声音，再看到来源 | 脚步声从远处接近 |
| 空镜恐惧 | 空无一人的画面暗示威胁 | 空荡荡的走廊，但门开着 |
| 日常异变 | 正常事物变得不正常 | 镜子里的倒影慢了一拍 |
| 禁忌打破 | 触犯规则后的后果 | "千万不要回头" |
| 慢逼近 | 威胁缓慢接近 | 背景中越来越近的身影 |

### 3.4 恐怖片节奏

```
日常 → 一丝不对劲 → 被忽视 → 不对劲加重 → 第一次惊吓 → 调查 → 更大的恐惧 → 至暗时刻 → 反击或逃跑 → 结局（通常是开放式）
```

### 3.5 中国恐怖片特殊规则

- 不能有真正的鬼（审查红线）
- 常用解释：精神疾病、梦境、幻觉、人为阴谋
- 技巧：暗示比展示更恐怖，让观众自己脑补
- 成功案例的心理恐怖路线：《隐秘的角落》式的日常异变

## 四、爱情

### 4.1 爱情片的核心结构

```
相遇 → 障碍 → 靠近 → 误会/分离 → 领悟 → 重逢/在一起
```

### 4.2 障碍设计

| 障碍类型 | 说明 | 示例 |
|---------|------|------|
| 身份障碍 | 社会地位/阶层差异 | 穷小子×富家女 |
| 关系障碍 | 已有伴侣/敌对家庭 | 罗密欧与朱丽叶 |
| 性格障碍 | 两人都有缺陷 | 两个控制狂 |
| 外部障碍 | 战争/疾病/距离 | 异地/战乱 |
| 误会障碍 | 信息差导致的误解 | 以为对方背叛 |
| 时间障碍 | 错过/重逢 | 多年后再遇 |

### 4.3 亲密场景写法

- **暗示 > 直接**——一个眼神、一次触碰、一个停顿，比直接写亲热更有力
- **情绪比动作重要**——观众在意的是角色的感受，不是做了什么
- **节奏控制**——慢→快→慢，呼吸感很重要
- **环境参与**——雨、风、光线、声音都是情绪的一部分

### 4.4 中国爱情剧市场

- 热门模式：甜宠、虐恋、先婚后爱、破镜重圆
- 平台差异：芒果TV偏好甜宠，爱奇艺偏好品质爱情
- 注意：情感尺度要符合审查，暧昧比露骨更安全

## 五、动作犯罪

### 5.1 动作场景原则

- **每场动作戏都要有剧情目的**——不是为打而打
- **空间感**——观众要知道谁在哪里、距离多远
- **物理规则**——建立的物理规则要一致（不能一会无敌一会弱鸡）
- **代价**——动作必须有后果（伤痛、疲惫、心理创伤）

### 5.2 动作场景节拍

```
1. 建立空间（观众知道战场在哪里）
2. 建立力量对比（谁强谁弱）
3. 第一轮交锋（试探）
4. 力量转换（弱者找到机会）
5. 高潮交锋（关键一击）
6. 后果（伤痛/代价/收获）
```

### 5.3 犯罪片结构模型

| 模型 | 说明 | 代表作 |
|------|------|--------|
| 猎人与猎物 | 警察追罪犯 | 《七宗罪》 |
| 犯罪过程 | 展示犯罪本身 | 《盗梦空间》 |
| 堕落之路 | 好人变坏 | 《绝命毒师》 |
| 赎罪 | 犯罪后的救赎 | 《肖申克的救赎》 |
| 黑色电影 | 道德灰色地带 | 《唐人街》 |

### 5.4 中国犯罪题材注意

- 审查要求：正义必须最终到来
- 犯罪手法不能过于详细（避免模仿）
- 暴力尺度：可以暗示，不宜过度血腥
- 成功案例：《隐秘的角落》《漫长的季节》——犯罪为壳，人性为核

## 六、奇幻科幻

### 6.1 世界观经济原则

- **规则要少**——建立3条核心规则，比建立30条规则更有效
- **规则要一致**——一旦建立就不能违反
- **规则要产生冲突**——好的规则本身就是戏剧冲突的来源
- **代价与限制**——超能力必须有代价，否则没有戏剧性

### 6.2 世界观构建清单

```
核心设定：这个世界与现实最大的区别是什么？
规则：这个设定带来什么规则？
限制：这个规则的边界在哪里？
代价：使用这个规则要付出什么代价？
社会影响：这个设定如何改变了社会结构？
视觉表现：观众如何"看到"这个世界的不同？
```

### 6.3 科幻类型光谱

```
硬科幻 ←──────────────────────→ 软科幻
│                                      │
│  《星际穿越》         《黑镜》        │
│  (科学驱动)           (人性驱动)      │
│                                      │
│  科学准确性高         科学是背景      │
│  观众需要耐心         观众门槛低      │
```

### 6.4 奇幻世界观的"经济性"

- 不要一次性解释所有设定——让观众通过角色的行动理解世界
- 用"新角色"作为观众的向导——让新角色问观众想问的问题
- 设定要服务于故事——如果一个设定对情节没有作用，删掉它

## 七、历史传记

### 7.1 真实事件改编的挑战

| 挑战 | 应对 |
|------|------|
| 观众知道结局 | 用过程的精彩弥补结局的已知 |
| 人物太复杂 | 选择2-3个核心特质放大 |
| 时间跨度大 | 选择关键时间段，而非全景式 |
| 史实与戏剧冲突 | 在不违背核心事实的前提下戏剧化 |
| 后人评价 | 避免"上帝视角"，保持人物的当下感 |

### 7.2 传记片结构选择

| 结构 | 说明 | 适用 |
|------|------|------|
| 编年体 | 按时间顺序 | 人物生平本身就有戏剧性 |
| 聚焦式 | 选择人生中最关键的一段 | 人物生平太长或太平淡 |
| 框架式 | 从一个"现在"的时间点回溯 | 需要反思视角 |
| 多视角 | 不同人讲述同一个人 | 人物有争议性 |

### 7.3 中国历史题材注意

- 重大历史事件不能戏说（审查红线）
- 历史人物需要尊重基本史实
- 可以"大事不虚，小事不拘"
- 古装剧需要考虑服饰、礼仪、语言的时代感
