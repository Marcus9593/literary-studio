# 通用写作技巧库

> 本文件包含适用于所有体裁的核心写作技巧。按需加载，不要在每次触发时自动读取。

---

## 一、人物塑造

- **冰山理论**：只展示十分之一，让读者感受水下的重量
- **矛盾法**：给人物一个与外表/身份相反的内在特质
- **选择法**：通过艰难选择揭示人物本质
- **对白塑造**：每个人物应有独特的说话方式（用词、节奏、口头禅）

## 二、情节设计

- **契诃夫之枪**：出现的元素必须被使用，被使用的元素必须提前出现
- **冲突层级**：内心冲突 → 人际冲突 → 社会冲突 → 命运冲突
- **悬念三要素**：提出问题、延迟回答、意外但合理的答案
- **节奏曲线**：紧张场景后给读者喘息，平淡中埋下暗涌

## 三、场景描写

- **五感法则**：至少调动三种感官
- **情绪映射**：环境反映或反衬人物心理
- **动态描写**：让静止的画面动起来
- **克制原则**：描写服务于叙事，不为炫技

## 四、对白技巧

- **潜台词**：人物说的和想的不一样
- **冲突对白**：每段对话都应有立场碰撞
- **信息控制**：对白传递信息但不像在"播报"
- **节奏变化**：长句短句交替，沉默也是对白

## 五、风格模仿

用户可能要求模仿特定作家或作品风格。处理方式：

1. 分析目标风格的核心特征（句式、用词、节奏、叙事视角、情感基调）
2. 提取可模仿的技巧要素
3. 在创作中融入这些要素，但保持故事原创性
4. 告知用户：风格模仿是技巧层面的学习，不是复制
